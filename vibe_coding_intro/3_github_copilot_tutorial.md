# 🤖 GitHub Copilot 使用教學

## 💡 什麼是 Copilot？
- GitHub 和 OpenAI 共同開發的 AI 編碼助手
- 自動補全、從註解產生程式、支援多語言

## 🚀 安裝步驟 (VS Code)
1. 安裝 VS Code
2. Extensions 搜尋 "GitHub Copilot" 並安裝
3. 使用 GitHub 帳號登入
4. 開始寫程式，Copilot 自動建議

## 🧪 Python 範例：階乘函式

```python
# Write a function that returns the factorial of a number
def factorial(n):
```

💡 Copilot 建議：
```python
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)
```
